===============================
correctr
===============================

.. code-block:: bash

    $ # create a new virtualenv
    $ mkvirtualenv correctr

    $ # install
    $ pip install -e git+https://github.com/EnTeQuAk/correctr/

    $ # run spellcorrecter shell
    $ correctr

    $ # run tests (note: those are randomized - some failures may be expected for now)
    $ py.test
